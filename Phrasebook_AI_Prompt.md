# Prompt สร้างคำศัพท์คลังศัพท์ (สำหรับป้อน AI)

ใช้กับ ChatGPT/Claude/Gemini ก็ได้ — แทนที่ `{{INPUT}}` ด้วยสถานที่หรือกลุ่มคำที่ต้องการ
แล้วคัดลอกผลลัพธ์ (TSV) ไปวางในช่อง "นำเข้าคำศัพท์" ของแอปได้ทันที

---

## แม่แบบ Prompt (คัดลอกทั้งบล็อกด้านล่าง)

```
คุณคือผู้ช่วยไกด์ทัวร์ ทำหน้าที่สร้าง "คลังคำศัพท์" ให้ไกด์ใช้อธิบายนักท่องเที่ยว
โจทย์ของฉันคือ: {{INPUT}}

จงสร้างรายการคำ/วลีที่เกี่ยวข้องกับโจทย์นี้ 15–25 คำ โดย "คำไทยเป็นต้นทาง" พร้อมคำแปลจีน+พินอิน และอังกฤษ

รูปแบบผลลัพธ์ (สำคัญมาก):
- ส่งออกเป็นตาราง TSV เท่านั้น (คั่นแต่ละช่องด้วย Tab) หนึ่งคำต่อหนึ่งบรรทัด
- 7 คอลัมน์ ตามลำดับนี้เป๊ะ:
  1) สถานที่        — ถ้าโจทย์เป็นสถานที่ ใส่ชื่อสถานที่; ถ้าเป็นกลุ่มคำทั่วไป เว้นว่าง
  2) คำ/วลี (ไทย)   — บังคับ
  3) คำแปลจีน       — อักษรจีนตัวย่อ
  4) พินอิน         — มีวรรณยุกต์ (เช่น Yùfó)
  5) คำแปลอังกฤษ
  6) หมวด L1        — เลือกจากรายการ L1 ด้านล่าง
  7) หมวด L2        — เลือกจากรายการ L2 ด้านล่าง

รายการหมวด L1 (เลือก 1): สถานที่ | อาหาร/ของกิน | วัฒนธรรม/ประเพณี | บุคคล/ประวัติศาสตร์ | การเดินทาง | ที่พัก | ช้อปปิ้ง/ของฝาก | สื่อสารทั่วไป | สุขภาพ/ฉุกเฉิน

รายการหมวด L2 (เลือกให้เข้ากับ L1):
- สถานที่: วัด/ศาสนสถาน, พระราชวัง/วัง, โบราณสถาน/ปราสาท, พิพิธภัณฑ์, อนุสาวรีย์/ศาล, ตลาด/ย่านเก่า/ชุมชน, ทะเล/ชายหาด/เกาะ, ภูเขา/น้ำตก/ถ้ำ, สวน/อุทยาน/ธรรมชาติ, จุดชมวิว/แลนด์มาร์ก
- อาหาร/ของกิน: อาหารคาว, ของหวาน/ขนม, เครื่องดื่ม/กาแฟ, ผลไม้, สตรีทฟู้ด/ของกินเล่น, วัตถุดิบ/เครื่องปรุง, รสชาติ/ความเผ็ด
- วัฒนธรรม/ประเพณี: ศาสนา/ความเชื่อ, พิธีกรรม/การไหว้, เทศกาล/งานประเพณี, ศิลปะ/หัตถกรรม, การแต่งกาย, มารยาท/ข้อควรปฏิบัติ
- บุคคล/ประวัติศาสตร์: พระมหากษัตริย์/ราชวงศ์, บุคคลสำคัญ/วีรบุรุษ, พระสงฆ์/นักบวช, เทพ/ตัวละครในตำนาน, ศิลปิน/ช่างฝีมือ, เหตุการณ์/ยุคสมัย
- การเดินทาง: ยานพาหนะ, ทิศทาง/บอกทาง, ป้าย/สัญลักษณ์, ตั๋ว/ค่าโดยสาร, เวลา/ตารางเดินทาง
- ที่พัก: ประเภทห้อง, สิ่งอำนวยความสะดวก, เช็คอิน/เช็คเอาท์, ปัญหา/การร้องขอ
- ช้อปปิ้ง/ของฝาก: ของที่ระลึก, การต่อราคา/ราคา, หน่วย/จำนวน, การชำระเงิน
- สื่อสารทั่วไป: ทักทาย/กล่าวลา, ขอบคุณ/ขอโทษ, ตัวเลข/การนับ, เวลา/วันที่, คำถามพื้นฐาน, อารมณ์/ความรู้สึก
- สุขภาพ/ฉุกเฉิน: อาการเจ็บป่วย, ยา/เภสัช, โรงพยาบาล/หมอ, เหตุฉุกเฉิน/ขอความช่วยเหลือ, ความปลอดภัย

ข้อห้าม:
- ห้ามใส่แถวหัวตาราง (header)
- ห้ามใส่คำอธิบาย ข้อความนำ ท้าย หรือ ``` ล้อมรอบ — ส่งเฉพาะบรรทัดข้อมูลล้วนๆ
- ห้ามใช้ Tab ภายในข้อความของช่อง (ถ้าคำมีหลายความหมายให้คั่นด้วย " / ")
```

## วิธีนำไปใช้

1. เปิด ChatGPT/Claude แล้ววางแม่แบบ แทน `{{INPUT}}` ด้วยโจทย์ (เช่น "ศัพท์ในโรงพยาบาล")
2. คัดลอกผลลัพธ์ทั้งหมด → วางในช่อง "นำเข้าคำศัพท์" ของแอป → กดนำเข้า
3. ตรวจ/แก้รายคำในแอปได้ภายหลัง (โดยเฉพาะพินอินและหมวด L2)

> เคล็ดลับ: ถ้าเครื่องมือ AI ทำให้ Tab เพี้ยนตอนคัดลอก ให้สั่งเพิ่มว่า "ส่งออกในกล่องโค้ด (code block) เพื่อรักษา Tab" แล้วคัดลอกจากในกล่องนั้น
